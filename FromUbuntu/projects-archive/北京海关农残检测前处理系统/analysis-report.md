## 2. 国标匹配分析

### 标准要求（GB23200.121-2021）
- **适用基质：** 蔬菜、水果、食用菌、谷物、油料、坚果、茶叶、香辛料
- **前处理方法：** QuEChERS
- **检测仪器：** HPLC-MS/MS

### 关键参数匹配
根据客户基质类型，需要支持两种处理流程：

**A. 蔬菜、水果、食用菌流程：**
1. 称样量：10g（干制样品 1g + 9mL 水）
2. 提取溶剂：10mL 乙腈
3. 盐包：4g 硫酸镁 + 1g 氯化钠 + 1g 柠檬酸钠二水合物 + 0.5g 柠檬酸二钠盐倍半水合物
4. 净化：每 mL 提取液用 150mg 无水硫酸镁 + 25mg PSA
5. 离心：4200r/min，5min
6. 过滤：0.22μm 微孔滤膜

**B. 谷物、油料和坚果流程：**
1. 称样量：5g
2. 水化：10mL 水，涡旋混匀，静置 30min
3. 提取溶剂：15mL 乙腈-乙酸溶液
4. 盐包：6g 无水硫酸钠 + 1.5g 乙酸钠
5. 净化：每 mL 提取液用 150mg 硫酸镁 + 50mg PSA + 50mg C18
6. 离心：4200r/min，5min
7. 过滤：0.22μm 微孔滤膜

## 3. 通量计算

### 客户要求
- 日送检样品数量：200个
- 单批次期望处理量：10个样本
- 每日工作时长：12小时
- 后端分析仪器配置：ICP-MS 4台

### 通量分析
1. **每日批次数：** 200个 ÷ 10个/批 = 20批/天
2. **批次间隔时间：** 12小时 ÷ 20批 = 36分钟/批
3. **全流程耗时估算（基于标准流程）：**
   - 样品称量：2分钟
   - 提取振荡：1分钟
   - 离心分离：5分钟
   - 净化涡旋：1分钟
   - 二次离心：5分钟
   - 过滤转移：2分钟
   - **总计：约 16分钟/批**
4. **通量可行性：** ✅ 可行
   - 理论批次时间（16分钟）< 可用批次间隔（36分钟）
   - 系统有足够的时间余量处理不同基质类型的差异
5. **后端仪器供应：**
   - 4台 ICP-MS，每台日处理能力约 50个样品
   - 系统日处理 200个样品，可满足 4台 ICP-MS 的供样需求
