## 1. 项目背景与需求分析

### 1.1 项目背景
随着工业化进程加快，城市河流重金属污染问题日益突出。为加强水环境监管，实现重金属污染的实时监测和预警，特建设本水质重金属在线监测系统。

### 1.2 客户需求
- **监测参数**：铅、汞、镉、铬等重金属指标
- **监测规模**：3个河流监测点
- **传输方式**：5G无线网络
- **系统平台**：Windows 11 Enterprise
- **数据存储**：5年实时数据保存
- **报警响应**：≤1小时
- **移动应用**：App监控硬件运行
- **系统对接**：环保局数据接口
- **报表输出**：每日Markdown格式报告

### 1.3 技术标准
- GB 3838-2002《地表水环境质量标准》
- HJ/T 91-2002《地表水和污水监测技术规范》
- GB/T 22239-2019《信息安全技术 网络安全等级保护基本要求》

## 2. 总体设计方案

### 2.1 系统架构
```
┌─────────────────────────────────────────────────────┐
│                水质监测云平台中心站                  │
│      (Windows 11 Enterprise + SQL Server 2022)      │
├─────────────────────────────────────────────────────┤
│ 数据接收 → 数据处理 → 实时监控 → 报警管理 → 报表生成 │
└─────────────────────────────────────────────────────┘
                    ↑ MQTT over 5G
┌─────────────────────────────────────────────────────┐
│                   5G无线传输网络                     │
└─────────────────────────────────────────────────────┘
                    ↑
┌───────────┐  ┌───────────┐  ┌───────────┐
│ 监测点1   │  │ 监测点2   │  │ 监测点3   │
│ (河流A)   │  │ (河流B)   │  │ (河流C)   │
└───────────┘  └───────────┘  └───────────┘
```

### 2.2 网络拓扑
```
监测点设备 → 5G路由器 → 互联网 → 防火墙 → 中心站服务器
                    ↓
            移动App/环保局系统
```

### 2.3 数据流设计
```
传感器采集 → 本地预处理 → 5G传输 → 中心接收 → 数据校验
    ↓                                        ↓
实时显示                                SQL Server存储
    ↓                                        ↓
阈值判断                                数据分析处理
    ↓                                        ↓
报警触发                                报表生成
```

## 3. 详细技术方案

### 3.1 硬件系统设计

#### 3.1.1 监测点设备配置
每个监测点包含：
1. **重金属在线监测仪**：WQMS-900HM XRF多参数监测系统
2. **自动采样单元**：恒流采样泵+预处理系统
3. **5G通信模块**：工业级5G路由器（华为/中兴）
4. **供电系统**：太阳能板+蓄电池+电源管理
5. **防护机柜**：防水防雷不锈钢机柜
6. **辅助设备**：温湿度传感器、视频监控

#### 3.1.2 中心站设备配置
1. **应用服务器**：戴尔PowerEdge R750（2台，主备）
2. **数据库服务器**：戴尔PowerEdge R750（SQL Server）
3. **存储系统**：戴尔PowerVault ME5系列（100TB）
4. **网络设备**：华为防火墙+交换机
5. **UPS电源**：山特20KVA在线式UPS
6. **监控大屏**：55寸拼接屏（3×3）

### 3.2 软件系统设计

#### 3.2.1 软件架构
- **操作系统**：Windows 11 Enterprise
- **数据库**：Microsoft SQL Server 2022
- **开发框架**：.NET 8.0 + ASP.NET Core
- **前端技术**：Vue.js + Element Plus
- **移动端**：React Native（iOS/Android）
- **通信协议**：MQTT 5.0 over 5G
- **报表引擎**：Markdown模板引擎

#### 3.2.2 功能模块设计

##### 1. 数据采集模块
- 实时接收3个监测点的重金属数据
- 数据格式校验和完整性检查
- 异常数据识别和标记

##### 2. 数据处理模块
- 数据清洗和标准化处理
- 单位换算和修约计算
- 数据质量评估

##### 3. 实时监控模块
- 监测点状态实时显示
- 重金属浓度趋势图
- 设备运行状态监控

##### 4. 报警管理模块
- 多级报警阈值配置
- 报警事件记录和追踪
- 多渠道报警通知（App、短信、邮件）

##### 5. 报表生成模块
- 每日自动生成Markdown格式报告
- 报告模板自定义
- 历史报告查询和导出

##### 6. 移动App模块
- 实时数据查看
- 设备远程控制
- 报警消息推送
- 报表查看和分享

##### 7. 系统管理模块
- 用户权限管理（RBAC）
- 系统日志审计
- 数据备份和恢复

##### 8. 对接接口模块
- 环保局数据上报接口
- 第三方系统集成接口
- API文档和测试工具

#### 3.2.3 数据库设计
- **实时数据表**：存储原始监测数据
- **报警记录表**：存储报警事件
- **设备状态表**：存储设备运行状态
- **用户日志表**：存储操作日志
- **报表数据表**：存储生成的报告
- **配置参数表**：存储系统配置

### 3.3 安全设计

#### 3.3.1 网络安全
- 防火墙隔离内外网
- VPN加密传输
- 入侵检测和防御
- 定期安全漏洞扫描

#### 3.3.2 数据安全
- 数据传输加密（TLS 1.3）
- 数据库访问控制
- 数据备份和容灾
- 操作日志审计

#### 3.3.3 应用安全
- 用户身份认证和授权
- 输入验证和防注入
- 会话管理和超时控制
- 安全编码规范
