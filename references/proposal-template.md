# 技术方案文档模板规范

> 此文件定义了百泉聚兴技术方案的标准章节结构和写作规范。

## 文档封面信息

```
Technical PROPOSAL

{检测领域}全流程自动化
样品前处理系统
技术方案

百泉聚兴（北京）科技有限公司
```

## 标准章节结构

### 1 系统简介
- 阐述系统背景和设计目的
- 说明参照的国家标准
- 概述自动化整合技术的优势
- 列出系统由哪些功能单元集成
- 给出系统效率概述（批次量、日处理量、可供应仪器数）

### 2 工艺流程
- 总体描述全流程自动化思路
- 按不同基质类型分别列出详细步骤
- 使用表格对比不同基质的处理参数差异
- 关键数值精确到标准要求的精度

### 3 系统组成
- 首先列出系统整体参数（电压/温度/湿度/尺寸/功率）
- 按编号逐个列出各功能单元
- 每个单元包含：功能描述 + 参数列表
- 参数必须完整，包含范围、精度、通道数等

### 4 软件系统
- 描述 SmartLabOS 的核心功能
- 方法库管理能力
- 追溯和报告功能
- 系统集成能力（LIMS/MES/WMS 如有需要）

### 5 系统效率分析
- 详细的通量计算过程
- 批次处理量 × 批次数 = 日处理量
- 与后端仪器的供给关系

### 6 现场要求
- 分小节列出：供电、供水、场地、环境要求
- 给出明确的数值要求

### 7 质量保证与服务
- 验收流程
- 质保期
- 服务响应承诺

## 写作规范

- 使用专业但清晰的中文
- 专业术语保持英文（QuEChERS、SPE、PSA、GCB 等）
- 数值使用国际单位制
- 参数列表使用编号格式
- 表格对齐规范
- 不使用口语化表达
