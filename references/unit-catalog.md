# 硬件功能单元目录

> ⚠️ 此文件列出所有可选的功能单元。请根据公司产品线补充完善。

## 农残/兽残前处理系统单元

| 单元名称 | 适用场景 | 关键参数 |
|---------|---------|---------|
| 加液涡旋单元 | 添加提取溶剂 | 5种液体，5μL-50mL，±1% |
| 加盐涡旋单元 | 添加无机盐包 | 0.5-20g，±0.5% |
| 振荡混合提取单元 | 样品提取 | 2通道，0-300rpm |
| 离心单元 | 固液分离 | 50/15mL管，最高10000rpm |
| 浓缩单元 | 溶剂浓缩 | 2位，45-50℃ |
| 移液单元 | 液体转移 | 0.1-10mL，<0.5% |
| 超声单元 | 超声辅助 | 40KHz，0-6通量 |
| SPE 单元 | 固相萃取 | 3cc/6cc柱，4种液体 |
| 加内标单元 | 添加内标 | 5-100μL，±2% |
| 过滤单元 | 滤膜过滤 | 0.22μm，13mm针式 |
