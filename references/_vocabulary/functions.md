# 工艺功能受控词表

> 模块卡片的 `function` / `sub_functions` 字段，以及平台卡片的 `supported_functions` 字段，**只能**从下表选取。
> 新增条目时先在此处登记，再在卡片中使用。

## 主功能 (function)

| 词条 | 含义 | 备注 |
|------|------|------|
| 称量 | 固体/液体定量称量 | |
| 加液振荡 | 添加溶剂并振荡混合 | 含加盐涡旋等子动作 |
| 提取 | 样品提取（含 QuEChERS 提取等） | |
| 萃取 | 液液萃取或固液萃取 | |
| 净化 | SPE / 净化盘 / 滤膜等净化操作 | |
| 浓缩 | 溶剂浓缩 | 含氮吹/旋蒸 |
| 浓缩定容 | 浓缩 + 定容 | 复合功能 |
| 定容 | 定量稀释/定容 | |
| 移液 | 液体精确转移 | |
| 过滤 | 滤膜过滤（针式/板式） | |
| 消解 | 微波消解/加热消解 | 无机元素前处理 |
| 离心 | 固液分离 | |
| 加热 | 恒温/加热 | 通常作为 sub_function |
| 冷却 | 恒温/冷却 | 通常作为 sub_function |
| 超声 | 超声辅助 | 通常作为 sub_function |

## 子功能 (sub_functions)

子功能可复用主功能词条；常见组合还包括：

| 词条 | 含义 |
|------|------|
| 加液 | 单纯添加液体 |
| 振荡 | 单纯振荡 |
| 涡旋 | 涡旋混合 |
| 加盐 | 添加无机盐包 |
| 加内标 | 添加内标液 |
| 留样 | 样品留存/分装 |

## 维护规则

- 新增/废弃词条时在表中追加，并在 `last_updated` 字段更新日期
- 废弃词条用 `~~删除线~~` 标注，而非直接删除（避免老卡片引用失效）
- 新增词条后跑一次 `python tools/build_indexes.py` 重建索引
