# 行业受控词表

> 方案卡片的 `industry` 字段**只能**从下表选取，用于 `solutions/_index.md` 的"按行业反查"分组。

| 词条 | 含义 | 典型场景 |
|------|------|---------|
| 食品检测 | 食品安全检测 | 农残、兽残、添加剂、营养成分 |
| 农产品质检 | 农产品产地/出口检测 | 海关、农科院 |
| 中药质量 | 中药质量分析 | 经典名方、饮片、中成药 |
| 制药 QC | 制药企业质控 | 化学药品、生物制品 |
| 环境监测 | 环境样品检测 | 水/土壤/大气 |
| 化工分析 | 化工产品分析 | 原料/中间体/成品 |
| 第三方检测 | 通用检测服务 | 综合性第三方实验室 |
| 海关通关 | 进出口检测 | 海关技术中心 |
| 食品检验院 | 各级食品检验机构 | 省/市食检院 |
| 疾控中心 | 公共卫生检测 | 省/市 CDC |
| 海关总署 | 海关总署直属 | 检疫科学技术研究中心 |

## 维护规则

- 行业划分以客户实际业务场景为准
- 一个客户可能跨多个行业（例如食品检测 + 海关通关），允许 `industry` 为列表
