# 样品类型受控词表

> 方案卡片的 `sample_types` 字段**只能**从下表选取。

## 食品/农产品

| 词条 | 含义 |
|------|------|
| 蔬菜 | 叶菜/根茎类等 |
| 水果 | 各类水果 |
| 食用菌 | 蘑菇/木耳等 |
| 谷物 | 稻麦/玉米等 |
| 油料 | 大豆/花生等 |
| 坚果 | 坚果类 |
| 茶叶 | 茶叶 |
| 香辛料 | 干香辛料 |
| 肉类 | 畜禽肉 |
| 水产品 | 鱼虾贝等 |
| 乳制品 | 奶/酸奶/奶粉 |
| 蛋类 | 蛋及蛋制品 |

## 中药 / 制药

| 词条 | 含义 |
|------|------|
| 中药材 | 原药材 |
| 中药饮片 | 切片饮片 |
| 中成药 | 制剂 |
| 经典名方 | 经方制剂 |
| 化学药品 | 原料药 / 制剂 |

## 环境 / 化工

| 词条 | 含义 |
|------|------|
| 水样 | 地表水/地下水/废水 |
| 土壤 | 土壤样品 |
| 大气颗粒物 | 滤膜采集 |
| 化工产品 | 化工原料/中间体 |

## 维护规则

- 新增样品类型先登记词表后使用
- 样品类型与国标"适用基质"对齐，更新国标时复核
